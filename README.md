# Сокращатель URL

Это простое приложение для сокращения URL.

## Установка

1. **Установите зависимости:**
   ```bash
   poetry install
   ```

2. **Запустите приложение:**
   ```bash
   poetry run uvicorn app.main:app --reload
   ```

## Тестирование

Для запуска тестов используйте следующую команду:
```bash
poetry run pytest
```
